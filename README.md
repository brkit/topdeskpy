## Help with finding ID's on topdesk
In the search menu of topdesk 
1. Select the type of ID you wanna find from the dropdown menu (eg. "Ansvarliggrupper")
2. Enter the name of the thing you want to find of
3. Select the desired item from the list (So that the row is "colored blue")
4. Move your mouse into the bottom right cornor of the window (A "π" symbol wil appear)
5. Rightclick the π and copy link address and the value of the "unid" in the URL parameter is the ID of your desired topdesk object
